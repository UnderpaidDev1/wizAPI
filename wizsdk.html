locate_on_screen(match_img, region=None, threshold=0.1, debug=False)¶

Attempts to locate match_img in the Wizard101 window. Returns (x, y) tuple for center of match if found. False otherwise. pass a rect tuple (x, y, width, height) as the region argument to narrow down the area to look for the image. Adjust threshold for the precision of the match (between 0 and 1, the lowest being more precise). Set debug to True for extra debug info
